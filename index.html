HTML

HTML é a linguagem padrão para criar páginas web.

Ele define a estrutura do conteúdo.

<html>

Dentro do elemento <html>, você pode encontrar os elementos <head> e <body>.

CSS

CSS é usado para descrever a apresentação de um documento escrito em HTML.

Ele controla a aparência do conteúdo.

body { background-color: #fff; }

JavaScript

JavaScript é uma linguagem de programação que permite a implementação de funcionalidades complexas em páginas web.

Ele permite que as páginas sejam interativas.
